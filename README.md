<h2>📌 Tentang Project</h2>

<p>
Project ini merupakan implementasi Business Intelligence menggunakan Python untuk melakukan pengolahan, analisis, dan visualisasi data.
Notebook ini dirancang untuk membantu proses pengambilan keputusan berdasarkan data melalui tahapan preprocessing, analisis data, hingga visualisasi informasi.
</p>

<p>
Tujuan utama project ini adalah menghasilkan insight bisnis yang dapat digunakan untuk mendukung evaluasi dan pengambilan keputusan secara lebih efektif.
</p>

<hr>

<h2>🎯 Tujuan Project</h2>

<ul>
  <li>Melakukan analisis data bisnis secara sistematis</li>
  <li>Membersihkan dan memproses dataset</li>
  <li>Menyajikan visualisasi data yang informatif</li>
  <li>Menghasilkan insight berdasarkan data</li>
  <li>Mendukung pengambilan keputusan berbasis data</li>
</ul>
